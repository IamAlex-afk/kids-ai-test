# Kids AI Test — Roadmap

## ФАЗА A: Рефакторинг сайта (4 этапа)

### Этап 1 — Разделение потоков (делаем сейчас)
- Стартовый экран: 2 гигантских карточки после boot()
  - "Я Родитель / Взрослый" → взрослая ветка
  - "Я Ребенок" → выбор возраста (tiny 3-6, child 7-12, teen 13+)
- Логика роутинга в core.js + CSS анимации

### Этап 2 — Ветка "Ребенок" / Tiny Mode (3-6 лет)
- Web Speech API: авто-озвучка вопросов
- UI: 2 ответа, 80% эмодзи, 20% текст, весь экран
- Auto-flow: без кнопки "Далее", пауза 2 сек после правильного → следующий

### Этап 3 — AI Дуэль (расширенная)
- Убрать SVG-роботов → <img> теги (WebP)
- Эволюция: каждые 5 правильных → более крутой робот
- Боссы каждые 10 уровней: ускорение таймера, другой визуал
- CSS анимации: атака (выпад), урон (тряска)
- Цель: с 5 раундов до 30

### Этап 4 — Виральность и монетизация
- Сертификат "AI-гений" + Web Share API
  - "Мой ребенок прошел AI Test и обошел 90% взрослых! Проверь себя: [ссылка]"
- Взрослая ветка: нативный блок (тёмная тема, неоновая рамка):
  - Книга "AI Biohacking: 33 Protocols" → https://www.amazon.it/dp/B0G35SBQR3
- /kids/ секция: отдельные SEO-страницы (AI Alphabet, Safety, Future Jobs и т.д.)

## ФАЗА B: Telegram Бот "AI Ментор для ребёнка" (следующий большой проект)

### Суть идеи
Родители в панике: дети деградируют на TikTok, ИИ — будущее.
Мы даём: Безопасный ИИ + Контроль + Результат.

### Что умеет бот
- Обучает (не просто отвечает) — промты настроены на школьную программу
- Расписание: ИИ "спит" когда ребёнок в школе, "просыпается" после
- Родители настраивают: предметы, уровень, время
- Промты в подарок при старте (готовые программы)
- Никаких карт, никаких данных — оплата через Telegram Stars

### Монетизация
- $10/месяц через Telegram Stars (без карты, без регистрации)
- Трафик: kids-ai-test → кнопка "Попробовать AI Ментора" → TG бот

### Технически
- Telegram Bot API + Claude API (claude-haiku-4-5 для дешевизны)
- Промты хранятся в конфиге, родитель выбирает из набора
- Расписание через cron/webhook по времени пользователя
- Деплой: Railway или Render (бесплатный tier для старта)

### Структура данных расписания
```json
{
  "child_name": "Саша",
  "age": 9,
  "school_hours": {"start": "08:00", "end": "14:00"},
  "active_topics": ["math", "english", "science"],
  "lesson_duration": 15,
  "language": "ru"
}
```

### Промты (пакеты в подарок)
1. "Объяснятор" — объясняет любую тему как учитель начальной школы
2. "Тренажёр" — задаёт вопросы по теме, ждёт ответ, хвалит/поправляет
3. "Читатель" — обсуждает книги из школьной программы
4. "ИИ-грамотность" — обучает безопасному использованию ИИ
